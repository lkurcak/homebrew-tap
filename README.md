# lkurcak tap

## How do I install these formulae?

`brew install lkurcak/tap/<formula>`

Or `brew tap lkurcak/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "lkurcak/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
